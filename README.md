# Songster

A starter Android app prototype for a **music timeline party game** inspired by the mechanics of Hitster.

## What this includes

- Kotlin + Jetpack Compose Android app scaffold.
- Basic `GameEngine` with demo song cards.
- Turn-based flow for 2 players.
- Timeline placement interactions (`place at start` / `place at end`) to validate core game loop quickly.

## Run locally

1. Install Android Studio (Jellyfish or newer).
2. Open this folder as a project.
3. Let Gradle sync.
4. Run `app` on an emulator or Android device.
